# Superstore Sales Dashboard

## Overview
An interactive retail analytics dashboard built using Power BI to analyze sales performance, profitability, customer trends, and business KPIs using the Superstore dataset.

This project transforms raw retail sales data into meaningful business insights through interactive dashboards and visual storytelling.

---

## Objectives
- Analyze overall sales and profit performance
- Identify top-performing and low-performing product categories
- Track regional sales trends
- Understand discount impact on profitability
- Create a decision-ready business dashboard

---

## Tools & Technologies
- Power BI
- Excel
- DAX
- Data Visualization
- Business Intelligence

---

## Key Insights
- High discounts reduced overall profitability in several categories
- Certain regions consistently outperformed others in sales
- A small number of products contributed significantly to total revenue
- Technology category generated strong profit margins compared to others
